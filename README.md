## Getting Started

### Prerequisites
- Node.js 18+
- Flutter 3.0+
- PostgreSQL 14+
- Redis 6+

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Configure your .env file
npm run dev
```

### Mobile Setup
```bash
cd mobile
flutter pub get
flutter run
```

### Web Setup
```bash
cd web
npm install
npm run dev
```
